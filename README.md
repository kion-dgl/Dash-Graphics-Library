### Install for Ubuntu, Debian

```
$ sudo apt-get install libXmu-dev libXi-dev libgl-dev dos2unix git wget libpng-dev
```

```
$ sudo yum install libXmu-devel libXi-devel libGL-devel dos2unix git wget libpng
```
