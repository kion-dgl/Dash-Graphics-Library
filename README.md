# DashGL
* Basic Vector and Matrix Manipulation Library
* Compatible with OpenGL
* Written in C
* MIT License

### Documentation

- wiki on github

### Tutorials

- view on dashgl.com

### Quick Start

- git clone
- cd
- make
- make install

### Dependencies for Ubuntu, Debian

```
$ sudo apt-get install libgtk-3-dev libegl1-mesa-dev libpng-dev
```
